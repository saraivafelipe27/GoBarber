    GoBarber API
</h1>

<h4 align="center">
  A barber scheduling app that allows users to schedule an appointment with his favorite barber and shows to the barber his agenda for the day.
</h4>

<h2 align="center">
  Instalação:
</h2>

<h3 align="center">
  yarn install
  yarn dev
</h3>
